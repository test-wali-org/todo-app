# Vanilla JS Todo App (Frontend + Express Backend)

Simple todo app with:
- Vanilla JS frontend
- Express backend API

## Features

- Add todos
- Mark todos complete/incomplete
- Delete todos
- Frontend calls backend routes under `/api/todos`

## Run locally

1. Install dependencies:
   - `npm install`
2. Start server:
   - `npm start`
3. Open:
   - `http://localhost:3000`
